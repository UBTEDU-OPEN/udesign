# Divider 分割线

区隔内容的分割线

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

## 基础用法

默认为水平分割线。

```jsx
<>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
  <Divider />
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
</>
```

  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>

## 包含内容

`children` 将作为内容显示到分割线上，默认居中显示，可通过 `orientation` 指定居左或者居右显示。

```jsx
<>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
  <Divider>Text</Divider>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
  <Divider orientation='left'>Text</Divider>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
  <Divider orientation='right'>Text</Divider>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
</>
```

## 边缘位置

可通过 `orientationMargin` 指定居左或者居右显示时距离边缘的距离。

```jsx
<>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
  <Divider orientation='left' orientationMargin={0}>
    Left Text with 0 orientationMargin
  </Divider>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
  <Divider orientation='right' orientationMargin={50}>
    Right Text with 50px orientationMargin
  </Divider>
  <p>这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案， 这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案这是一段待编辑的文案。</p>
</>
```

## 垂直分割线

可通过 `vertical` 设置为垂直分割线。

```jsx
<>
  Text
  <Divider vertical />
  <a href='#'>Link</a>
  <Divider vertical />
  <a href='#'>Link</a>
</>
```
