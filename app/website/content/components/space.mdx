# Space 间距

设置组件之间的间距。

## 何时使用

- 避免组件紧贴在一起，拉开统一的空间。
- 适合行内元素的水平间距。
- 可以设置各种水平对齐方式。

## 基础用法

相邻元素水平间距。

```jsx
<Space>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Space>
```

## 垂直间距

通过 vertical 设置为垂直间距模式。

```jsx
<Space vertical align='start'>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Space>
```

## 间距大小

通过 size 设置尺寸，默认为 small，可选 large 和 middle。

```jsx
<Space size='small'>
  small
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Space>
```

```jsx
<Space size='middle'>
  middle
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Space>
```

```jsx
<Space size='large'>
  large
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Space>
```

## 对齐方式

通过 align 设置对齐方式，类似 flex 布局的 align-items。

```jsx
<Space>
  <Space className='border p-3 rounded' size='small'>
    center
    <div className='px-5 py-3 bg-indigo-500 text-white rounded'>Primary</div>
    <div className='px-5 py-10 bg-indigo-50 rounded'>Block</div>
  </Space>
  <Space className='border p-3 rounded' align='start' size='small'>
    start
    <div className='px-5 py-3 bg-indigo-500 text-white rounded'>Primary</div>
    <div className='px-5 py-10 bg-indigo-50 rounded'>Block</div>
  </Space>
  <Space className='border p-3 rounded' align='end' size='small'>
    end
    <div className='px-5 py-3 bg-indigo-500 text-white rounded'>Primary</div>
    <div className='px-5 py-10 bg-indigo-50 rounded'>Block</div>
  </Space>
  <Space className='border p-3 rounded' align='baseline' size='small'>
    baseline
    <div className='px-5 py-3 bg-indigo-500 text-white rounded'>Primary</div>
    <div className='px-5 py-10 bg-indigo-50 rounded'>Block</div>
  </Space>
</Space>
```

## 自定义尺寸

通过向 size 传入 number，来自定义间距大小。

```jsx
() => {
  const [size, setSize] = useState(10);

  return (
    <Space size={size} wrap>
      <Button>Button</Button>
      <Button>Button</Button>
      <Button>Button</Button>
      <Input defaultValue={String(size)} onChange={(value) => setSize(Number(value))} style={{ width: 150 }} />
    </Space>
  );
};
```

## 自动换行

通过 wrap 设置是否自动换行，默认 false。

```jsx
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>

  <Button>Button</Button>

  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
  <Button>Button</Button>
</Space>
```
