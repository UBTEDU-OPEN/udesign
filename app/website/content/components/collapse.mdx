# Collapse 折叠面板

可以折叠/展开的内容区域。

## 何时使用

对复杂区域进行分组和隐藏，保持页面的整洁。手风琴是一种特殊的折叠面板，只允许单个内容区域展开。

## 基础用法

为了适用更多的场景，没有样式，需要用户自行样式化。

## 受控面板

Collapse 的值由 activeKey 控制，配合 onChange 使用。如果不希望使用受控组件，请使用 defaultActiveKey。

```jsx
() => {
  const [activeKey, setActiveKey] = useState('1');
  return (
    <>
      <style>
        {`
          .controller .ud-collapse-item-header{
              background: #fff;
              border-radius: 0px;
              margin-bottom: 10px;
              padding: 22px 40px;
          }
          .controller .ud-collapse-item p{
              padding: 22px 40px;
              background: #EDF1FF;
          }
          .controller .ud-collapse-item.ud-collapse-item-border-active {
              background: transparent;
              border:none
          }

      `}
      </style>
      <Collapse
        className='controller'
        activeKey={activeKey}
        closeIcon={<DownOutlined />}
        expandIcon={<UpOutlined />}
        onChange={(val) => {
          if (val === activeKey) {
            setActiveKey('');
            return;
          }
          setActiveKey(val);
        }}
      >
        <Collapse.Item title='这里是标题1' name='1'>
          <p>
```
