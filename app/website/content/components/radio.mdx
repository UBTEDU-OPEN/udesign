# Radio 单选框

用于在选中和非选中状态之间进行切换。

## 基础用法

默认的选框

```jsx
<Radio>单选框</Radio>
```

## 禁用状态

通过 disabled 属性设置禁用状态。

```jsx
<Space>
  <Radio disabled>单选框</Radio>
  <Radio defaultChecked disabled>
    单选框
  </Radio>
</Space>
```

## 受控组件

当传入 checked 时，输入框将变成受控组件，必须配合 onChange 回调函数使用。如果不希望使用受控组件，请使用 defaultValue。

```jsx
() => {
  const [checked, setChecked] = useState(false);
  return (
    <Space vertical align='start'>
      <Button onClick={() => setChecked(!checked)}>{checked ? 'unchecked' : 'checked'}</Button>
      <Radio checked={checked}>单选框</Radio>
    </Space>
  );
};
```

## 单选组合（jsx）

通过在 Radio.Group 内部放置 Radio 元素，可以声明 Radio 组

```jsx
<Space>
  <Radio.Group name='group' defaultValue='2'>
    <Radio value='1'>Option A</Radio>
    <Radio value='2'>Option B</Radio>
    <Radio value='3'>Option C</Radio>
  </Radio.Group>
</Space>
```

## 单选组合（数组）

通过 options 属性生成 Radio 组

```jsx
<Radio.Group options={['A', 'B', 'C']} name='group' defaultValue='A' />
```

## 垂直排列

搭配 Space 组件可以实现单选组的垂直排列。

```jsx
<Radio.Group name='group' defaultValue='2'>
  <Space vertical>
    <Radio value='1'>Option A</Radio>
    <Radio value='2'>Option B</Radio>
    <Radio value='3'>Option C</Radio>
    <Radio value={'more'}>more...</Radio>
  </Space>
</Radio.Group>
```

## 按钮样式

通过 Radio.Button 使用按钮样式的单选框。

```jsx
<Space>
  哪一款产品最好看？
  <Radio.Group name='group' defaultValue='a'>
    <Radio.Button value='a'>uCode</Radio.Button>
    <Radio.Button value='b'>uDesign</Radio.Button>
    <Radio.Button value='c'>uExplore</Radio.Button>
  </Radio.Group>
</Space>
```
