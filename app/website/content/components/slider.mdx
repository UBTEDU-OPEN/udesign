# Slider 滑动输入条

## 基础用法

默认的滑动输入条。

```jsx
<Slider />
```

## 禁用状态

通过 disabled 属性设置禁用状态。

```jsx
<Slider disabled />
```

## 尺寸

通过 size 设置尺寸，默认 middle，可选 small。

```jsx
<Slider />
```

```jsx
<Slider size='small' />
```

## 滑动范围

通过 min 设置最小值，默认 0；通过 max 设置最大值，默认 100。

```jsx
<Slider max={10} min={0} />
```

## 受控属性

Slider 的值由 value 控制，配合 onChange 使用。如果不希望使用受控组件，请使用 defaultValue。

```jsx
<Slider value={50} />
```

```jsx
() => {
  const [value, setValue] = useState(50);
  return (
    <Slider
      value={value}
      onChange={(v) => {
        setValue(v);
        Toast(`${v}`);
      }}
    />
  );
};
```

## 回调

使用 onAfterChange 事件，触发时机与 onmouseup 一致。

```jsx
<Slider
  onAfterChange={(e) => {
    Toast(`${e}`);
  }}
/>
```
